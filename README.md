# 🎮 Catálogo de Jogos

Aplicativo desenvolvido em Flutter para cadastro e gerenciamento de jogos.

## 📱 Descrição

O aplicativo permite ao usuário criar, visualizar, editar e excluir jogos, funcionando como um catálogo pessoal.

Foi desenvolvido com foco em conceitos básicos de Flutter, como navegação entre telas, uso de widgets Stateful e Stateless, e manipulação de listas em memória.

---

## 🚀 Funcionalidades

- 📋 Listar jogos cadastrados
- ➕ Adicionar novos jogos
- ✏️ Editar jogos existentes
- 🗑️ Excluir jogos
- 🔍 Visualizar detalhes de um jogo

---

## 🧩 Dados do jogo

Cada jogo pode conter:

- Nome
- Gênero
- Plataforma
- Ano de lançamento (opcional)
- Status (Jogando, Zerado, Quero jogar)
- Nota (somente para jogos já jogados)
